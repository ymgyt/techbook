# cargo atcoder

## install

```sh
cargo install cargo-atcoder
```

## usage

```sh
# login
cargo atcoder login

# create contest directory
# https://atcoder.jp/contests/abc123
cargo atcoder new abc123

# submit
# cd abc123
# https://atcoder.jp/contests/abc123/tasks/abc123_b
cargo atcoder submit b
```
