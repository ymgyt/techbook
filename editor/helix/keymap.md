# Keymap

https://docs.helix-editor.com/keymap.html

## Navigation

### Select word

* w/W: before begiinng of the next word
* e/E: endo of the current word
* b/B: backword to begin of the word

### Select line

* x: select current line

### Select words

* s: x等でselectした状態でs押してwordいれてenter

### Deselect

* ;: selectionを解除する

## Manipulation

* u: undo
* U: redo

## Mode

* I: 行の先頭にinsert
* A: 行の末尾にappend
