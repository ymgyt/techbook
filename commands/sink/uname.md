# uname

systemに関する情報を表示

```sh
# release
uname -r
```

| options | description     | example               |
|---------|-----------------|-----------------------|
| `-r`    | Current release | …                     |
| `-m`    | Machine         | …                     |
| `-s`    | Kernel          | Linux,Darwin, FreeBSD |
| `-o`    | OS              | …                     |
