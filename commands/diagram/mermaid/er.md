# ER diagram

```
%%{init: {
    "er": {
        "layoutDirection": "TB",
        "entityPadding": 15,
        "useMaxWidth": true,
    }
}}%%
```

* `layoutDirection` layoutの方向性
    * `TB`: top to bottom. `BT`: bottom to top.
    * `LR`: left to right. `RL`: right to left.

* `entityPadding`: わかっていない
* `useMaxWidth`: 巨大なdiagramの場合はfalseにした方が良い?
