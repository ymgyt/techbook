# S3

## Lifecycle

> S3 Lifecycle configuration is a set of rules that define actions that S3 applies to a group of objects.

actionには2種類ある。

* Transition actions  
  * 30日たったら、Glacierに移すなんかができる。
* Expiration actions
  * 30日たったら削除するとか。
