# rust-analyzer

## Install

```shell
# from source
git clone https://github.com/rust-lang/rust-analyzer.git && cd rust-analyzer
cargo xtask install --server
```

## Configuration

先頭の`rust-analyzer.`は省略している。  

https://rust-analyzer.github.io/manual.html#configuration

* `cargo.features` (default `[]`)
  * 有効にするfeatures
