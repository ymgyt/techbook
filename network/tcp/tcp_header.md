# TCP Header

0                   15 16                                  31
+---------------------+--------------------------------------+
| Source Port         | Destination Port                     |
+---------------------+--------------------------------------+
| Sequence Number                                            |
+------------------------------------------------------------+
| Acknowledgment Number                                      |
+------------------------------------------------------------+
| Data Offset | Flags | Window Size                          |
+------------------------------------------------------------+
| Checksum              | Urgent Pointer                     |
+------------------------------------------------------------+
| Options (optional) ...                                     |
+------------------------------------------------------------+
| Payload (data) ...                                         |
+------------------------------------------------------------+


* optionがなければ、TCP Headerは20byte

* Data Offset: Headerの長さ.optionがあるのでHeaderが可変長

## TCP Flags

* SYN
* ACK
* FIN
* RST
* PSH
* URG

## Window size

受信側の空きbuffer. `rwnd`に使われる
