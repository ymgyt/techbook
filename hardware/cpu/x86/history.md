# x86の歴史

* 8086
  * 1978年 16bit
  * 20bit物理アドレス空間
    * 物理 = (segment << 16) + offset
  * リアルモードのみ

* 80286
  * 1982年 16bit
  * 保護モード(GDT/LDT 特権レベル)
  * 24bit物理アドレス空間

* 80386(IA-32)
  * 1985年 32bit
  * paging(仮想記憶,特権リング)導入

* 80486/Pentium
  * 1989年 32bit

* Pentium4
  * 1995 32bit

* AMD64
  * 2003年 64bit
