# typst

## Install

```sh
brew install typst
```


## Reference

https://typst.app/docs/reference/


## Usage

```sh
# create note.pdf
typst note.typ 

# watch
typst --watch note.typ
```

## Syntax

```
= Heading
content xyz

== Heading 2

+ item 1
  - bullet item 1-1
  - bullet item 1-2
+ item 2
+ itme 3
```
