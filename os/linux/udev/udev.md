# udev

* `/dev`のdevice file(node)を動的に管理する userspace process
  * `systemd-udevd.service`

* kernel内でdevice関連のeventが発生すると、netlink経由でuser spaceのudev daemonに通知が届く

* event処理
  * `/sysfs`を読む
  * ruleは`/etc/udev/rules.d`に定義する
    * `/usr/lib/udev/rules.d`
